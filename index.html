🚀 Linux Tutorial: Learn the Basics

📁 Navigating Directories

cd: Change directory. cd .. moves up one level.

📍 Print Current Directory

pwd: Displays the full path of your current working directory.

📄 Listing Files

ls: Lists files. ls -l: Long format. ls -a: Show hidden files.

📝 Creating & Editing Files

touch file.txt: Creates file. Use nano, vim, or gedit to edit.

📂 Creating Directories

mkdir folder: Makes new folder. mkdir -p parent/child: Nested folders.

🗑️ Deleting Files & Folders

rm file: Deletes file. rm -r folder: Deletes folder recursively.

📋 Copying & Moving Files

cp: Copy files. mv: Move or rename files/folders.

🔍 Finding Files

find . -name "file.txt": Searches file from current directory.

🔎 Viewing File Content

cat: Show full file. less, head, tail: View parts.

🔐 File Permissions

chmod: Change file permissions. chown: Change ownership.

👤 User Management

adduser: Create user. passwd: Set password. deluser: Remove user.

📦 Installing Packages

sudo apt install: Install on Ubuntu/Debian. Use dnf or yum for RedHat.

🧪 Running Scripts

bash script.sh: Run script. Or: chmod +x & ./script.sh

🖥️ Monitoring System

top: Process viewer. df -h: Disk usage. free -m: Memory info.

📡 Network Commands

ping: Test connectivity. ip a: IP info. netstat -tuln: Open ports.

📜 Viewing Logs

tail -f /var/log/syslog: Follow system logs live. Replace with any log file.

📌 Process Management

ps aux: List processes. kill PID: Terminate process.

💡 Terminal Shortcuts

Ctrl + C: Cancel command. Ctrl + D: Logout. Ctrl + L: Clear screen.
